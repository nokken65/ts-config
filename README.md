# `@nokken65/ts-config`

> Here is my personal [TypeScript](https://www.typescriptlang.org/) config.

## Install

```sh
npm install @nokken65/ts-config -D
```

```sh
yarn add @nokken65/ts-config -D
```

```sh
pnpm add @nokken65/ts-config -D
```

## Usage

```js
// tsconfig.json
{
  "extends": "@crashmax/tsconfig",
  "compilerOptions": {
    "outDir": "dist"
  },
  "include": [
    "src"
  ]
}
```
